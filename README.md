# 职工管理系统（基于C++多态实现）

这是一个使用 C++ 面向对象编程思想（封装、继承、多态）开发的**职工信息管理控制台程序**，支持对 Boss、Manager、Employee 三类职工进行增删改查操作，并通过文件读写实现数据持久化。

## ✨ 核心功能
*   **职工管理**：支持三类职工（Boss/Manager/Employee）的**增加、删除、修改、查询、显示**。
*   **多态展示**：通过虚函数 `showInfo()` 实现不同类型职工信息的差异化输出。
*   **数据持久化**：通过文件流 (`fstream`) 将职工信息保存至 `empFile.txt`，实现程序重启后数据不丢失。
*   **控制台界面**：清晰的菜单驱动交互，易于操作。

## 🔧 技术要点
*   **面向对象设计 (OOP)**
    *   **封装**：将职工信息与行为封装在类中。
    *   **继承**：定义 `Worker` 基类，派生出 `Boss`、`Manager`、`Worker` 子类。
    *   **多态**：基类定义虚函数，子类重写，实现运行时动态绑定，统一接口处理不同职工。
*   **文件操作 (File I/O)**
    *   使用 `fstream` 进行数据序列化与反序列化。
*   **代码组织**
    *   头文件与源文件分离，结构清晰。
    *   通过 `WorkManager` 类统一管理业务逻辑。

## 🚀 如何运行
1.  **环境**：任何支持 C++ 的 IDE 或编译器（如 Dev-C++, VS Code, Code::Blocks 等）。
2.  **编译运行**：
    *   使用 **Dev-C++**：直接打开 `EmployeeSystem1.dev` 项目文件，编译运行即可。
    *   使用其他编译器：将所有 `.cpp` 和 `.h` 文件添加到项目中，编译运行 `main1.cpp`。
3.  **使用**：根据控制台菜单提示，输入对应数字键执行功能。
